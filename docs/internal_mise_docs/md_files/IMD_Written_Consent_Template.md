## INSTRUCTIONS FOR USE

### When to Use This Template

Use this template when the Board of Directors needs to authorize actions requiring formal board approval:

| Action Type | Examples |
|-------------|----------|
| **Contracts** | Approving significant agreements |
| **Equity** | Stock grants, option grants, SAFE agreements |
| **Financing** | Loans, credit lines, investment rounds |
| **Officers** | Appointments, removals, compensation |
| **Corporate** | Bylaw amendments, committee formation |
| **M&A** | Acquisitions, significant asset transactions |
| **Ratification** | Approving prior actions |

### How to Complete

1. Replace `/TextEffectiveDate/` with the effective date
2. Replace `/TextResolutionTitle/` with a descriptive title
3. Fill in `/TextBackground/` with relevant context
4. Draft specific resolutions in `/TextResolution1/` and `/TextResolution2/`
5. Ensure all directors sign and date
6. File the executed consent in corporate records

### Filing and Recordkeeping

> → Original signed consents: Corporate minute book
> → Digital copies: `/Docs/Legal/Board Consents/`
> → Filename format: `YYYY-MM-DD_[Brief_Description].pdf`
