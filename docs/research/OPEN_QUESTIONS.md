## About This File

This file tracks questions that matter but do not yet have answers. Its purpose is intellectual honesty: to make visible the things we do not know, so that research can be directed toward the gaps that matter most.

This file is expected to grow and change. Questions will be added as research surfaces them. Some will be answered and marked resolved. Others may persist for months.

Not all questions need immediate answers. Some are worth holding open because the act of noticing the question is itself valuable â€” it prevents premature certainty.

## Entry Format

Each question follows this structure:

```
### Q-[number]: [The question, stated directly]

**Why this matters:** [1-2 sentences on why Mise should care about the answer]

**Related domains:** [e.g., payroll, inventory, ops, AI, GTM, hiring, legal, product]

**Possible next steps:** [What research or investigation might move toward an answer]

**Status:** [Open / In Progress / Resolved]

**Date opened:** [YYYY-MM-DD]

**Date resolved:** [YYYY-MM-DD, if applicable]

**Resolution:** [Brief summary of the answer, if resolved]
```
